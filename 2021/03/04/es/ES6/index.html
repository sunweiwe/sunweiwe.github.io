bind

bind() 方法会创建一个新函数。当这个新函数被调用时，bind() 的第一个参数将作为它运行时的 this，之后的一序列参数将会在传递的实参前传入作为它的参数。(来自于 MDN )

特点

  • 返回一个函数
  • 可以传入参数

call

call() 方法在使用一个指定的 this 值和若干个指定的参数值的前提下调用某个函数或方法
Array.prototype.slice.call(Object,1) 将 slice 绑定到 Object 上调用

apply

call()方法的作用和 apply() 方法类似，区别就是 call()方法接受的是参数列表，而 apply()方法接受的是一个参数数组。(MDN)

构造函数创建对象

每个函数都有一个 prototype 属性指向了一个对象，这个对象正是调用该构造函数而创建的实例的原型.

instance

instanceof 运算符用来检测 constructor.prototype 是否存在于参数 object 的原型链上。

typeof

typeof 操作符返回一个字符串，表示未经计算的操作数的类型。

instance compare with typeof

作用点的不同

  • typeof 主要用来判断基础数据类型，instanceof 则是用来判断引用数据类型。

底层逻辑的不同

  • typeof 是根据数据在存储单元中的类型标签来判断数据的类型，数据存储都使用二进制码，不同的数据类型的二进制码前三低位代表其类型。instanceof 则是根据函数的 prototype 属性值是否存在于对象的原型链上来判断数据的类型。
