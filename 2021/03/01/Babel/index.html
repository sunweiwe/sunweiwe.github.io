导入模块

webpack 打包分析

  • babel 将 import 替换成 require
  • webpack 将 require 替换成 __webpack_require__
  • __webpack_require__.r 用于标记 ES 模块，修改了 exports 的 toString 方法，并添加了一个 esModule 成员。
  • __webpack_require__.o hasOwnProperty shorthand
  • __webpack_require__.u get javascript chunk filename
  • __webpack_require__.l 通过 script 标签加载 chunk

入口方法

使用 __webpack_require__ 导入直接 import 的模块

  • 根据文件的相对路径查找
  • 如果有缓存直接返回缓存中，没有就到__webpack_modules__中查找并返回

使用 __webpack_require__.e 加载动态 import() 的模块

  • 核心就是将 require.ensure 转化为模拟 jsonp 去加载目标 chunk 文件。
